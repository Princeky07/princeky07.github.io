HTML Headings(this is H1)

  • To create a heading in an HTML document you use the h1 to h6 tags.
  • h1 has the largest text and h6 has the smallest text.
  • NOTE! Dont use headings to just make text big.
  • Use The heading tags for the intended purpose, Headings.
  • Search engines use the headings to structure the content of your web pages.
  • you can edit the heading size with a style attribute if needed

HTML Paragraphs

To begin a Paragraph you will use the p tag.
Any and all content you wish to show in normal sizing will go in the paragraph tags.
The browser will ignore spaces in code.

the    pre    tag    will   save   spaces   .
